# Xtra
Xtra Php Classes

### Zaktualizuj zależności (dev)
```bash
composer update
```

### Dodaj do swojego projektu
```bash
# Zobacz, wyszukaj
composer show moovspace/xtra
composer search moovspace/xtra

# Dodaj
composer require moovspace/xtra:~1.0

# Dodaj do composer.json
{
    "require": {
        "moovspace/xtra": "~1.0"
    }
}
```
